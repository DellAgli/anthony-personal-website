TwitchWall

TwitchWall is a companion site for the videogame livestreaming site Twitch.tv. It allows the user to easily watch multiple livestreams all on the same page. Unlike other sites, it has the ability to automatically fill out the page with livestreams based on how many other people are currently watching.
